phpqrcode Library — Required for QR Code Generation
=====================================================

QUICK INSTALL
-------------
1. Go to: https://sourceforge.net/projects/phpqrcode/files/
2. Download phpqrcode-2010120ists.zip (or latest)
3. Unzip it — copy ONLY qrlib.php into this folder:
   wp-content/plugins/wc-ticket-qr/lib/phpqrcode/qrlib.php

That single file is all that is needed.

If qrlib.php is not present, the plugin generates QR codes
as SVG (works fine for scanning) as a fallback.
